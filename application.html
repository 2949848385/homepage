白名单申请要求及审核标准 最后更新时间: 2014-06-06

我们相信您已经满怀期待地想要加入我们的游戏。为了您能够更加顺利地通过白名单审核，请仔细阅读申请要求及审核标准。

申请要求

  1. 申请须在社群新人报到区发表申请帖，请包含以下内容
    • 玩家游戏 ID (后期可能更改为 GameProfile ID)
    • 受邀途径，包含以下情况
      • 在某社区/论坛看到宣传帖，附带帖子地址
      • 由我们的玩家邀请而来，并 mention 对应玩家，相应玩家也应回复以示确认邀请
      • 旧世界老玩家，直接 mention +Whale Mo 要求入服
    • 一句话自我介绍，不可以是空泛无意义的话
    • 如果没有通过审核，您的想法或行动
  2. 个人主页地址，目前我们接受的审核地址包括(好感度权重依次降低)
    • Github
    • Google+
    • 个人网站(您的个人博客/页面或与其他人合作运营的公共站点，如是后者请说明您的职位)
    • Twitter
    • Facebook
    • 人人网
    • 新浪微博
  3. 如果没有可用的个人主页，则必须更详细地介绍自己。个人隐私问题可以回避或解释说明，但是必须包含以下几点
    • 接触 Minecraft 时间，是否正版用户
    • 以前在哪些服玩过，或者在单机游戏，有没有什么作品
    • 游戏中印象深刻的几个记忆片段
    • 个人喜好，擅长的事情(任何擅长均可)以及性格
    • 目前在做的事情(学生、在读年级，已工作、从事的职业等)

审核标准

  1. 绝对禁止的信息：一旦发现此类信息，申请用户将直接被列入黑名单且不会解封
    • 色情、暴力等令人厌恶的信息
    • 反动政治、邪教等信息
    • 自发或转发的传销信息
    • 恶语伤人、出言不逊等极其不礼貌、不尊重他人的言论/行为
  2. 不良信息：此类信息将降低管理员的好感度，不利于通过审核
    • 伸手党及疑似行为
    • 过多的牢骚或抱怨
    • 优越感爆棚
    • 抖S或抖M或经常NTR
  3. 加分项：此类信息可以为您的申请加分，更容易通过审核
    • 有较好的 Minecraft 作品或其他ACG类作品
    • 活跃于开源社区
    • 喜欢折腾新鲜技术，但至少对一种技术有恒心和毅力
    • 写过开源的程序，或对其他开源项目贡献过代码/文档
    • 比较萌
  4. 直接通过：包含以下几种情况
    • 您是我们所熟知的人，我们确定您有资格且被欢迎加入游戏
    • 您在技术界/ACG圈有足够的影响力和正面声誉，且我们认为您能够符合一般玩家准则

审核细则
根据您提供的信息及以上审核标准，我们需要确定您满足以下所有要求。

  • 了解社区礼仪，尊重他人
  • 有足够的自制能力，能够在游戏里做到主动遵守规则
  • 有能力充分理解我们的游戏规则并应用到游戏中
  • 能够在游戏中主动承担起相应的玩家责任，在发生问题或冲突时，主动承担自己方面的责任并协助管理员解决问题
  • 和我们的主题有交集，能够被我们的玩家群体所接受
  • 心理健康，乐观且积极向上，不会轻易用恶意来揣测他人，更不会心怀鬼胎
  • 心智成熟，没有严重的性格和智力缺陷，不会对其他玩家造成困扰
在您提交白名单申请后，我们会在72小时内给您答复。如果申请通过，请在通过后24小时内登录服务器注册。
